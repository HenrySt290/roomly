# My Flutter App

A new Flutter project for Android development.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK (installed at `/mnt/oss/flutter`)
- Android SDK (needs to be installed)

### Running the app

To run the app on an Android emulator or device:

```bash
export PATH="$PATH:/mnt/oss/flutter/bin"
flutter run
```

### Building APK

To build a release APK:

```bash
export PATH="$PATH:/mnt/oss/flutter/bin"
flutter build apk --release
```

## Project Structure

- `lib/main.dart` - The main entry point of the app
- `pubspec.yaml` - Project configuration and dependencies
- `analysis_options.yaml` - Linter rules
